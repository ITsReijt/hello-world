# Java example

A short README that documents the Java HelloWorld example included in this repository.

Location

- src/HelloWorld.java

Requirements

- Java JDK 8 or newer

How to compile and run

```bash
javac src/HelloWorld.java
java -cp src HelloWorld
```

Expected output

```
Hello, World!
```

Notes

- The example is intentionally simple and has no package declaration so it runs from the src directory.
- See CONTRIBUTING.md for contribution guidelines.
