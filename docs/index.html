This simple Pages site lives in the docs/ folder and shows how to run the Java example that ships with this repository.

Run the Java example

From the repository root, compile and run the single-file example:

javac src/HelloWorld.java
java -cp src HelloWorld

Expected output: Hello, World!

Notes & next steps

  • The example uses no package declaration so it runs directly from src/.
  • Consider adding a Gradle or Maven project for more Java examples (I can help).
