# Contributing

Thanks for your interest in contributing! This repository is a personal learning project; contributions and suggestions are welcome.

How to contribute

1. Fork the repository.
2. Create a new branch for your change (feature/bugfix):
   - git checkout -b feature/my-change
3. Make your changes, commit with a clear message.
4. Push the branch and open a pull request against `main`.

Coding style

- Keep examples small and focused.
- If adding code, include a brief README or comments explaining how to run it.

Running the Java example

From the repository root:

```bash
javac src/HelloWorld.java
java -cp src HelloWorld
```
